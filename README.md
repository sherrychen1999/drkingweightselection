# 金鶯診所｜減重特定申報資料篩選器

一個瀏覽器小工具，用來協助 13 院區的櫃檯人員快速處理「特定申報」XLSX 檔案：自動篩選減重相關資料、計算人次與週次、寫入 Google Sheet 資料庫，並提供跨院區、跨醫師、跨雙週的儀表板與匯出 Excel 功能。

## 線上使用

直接打開：**https://你的帳號.github.io/這個repo/**

（GitHub Pages 啟用後，把上行的網址改成自己的）

## 功能特色

- **拖檔即用**：把 R1110 匯出的 XLSX 拖進去就自動篩選、上傳
- **批次處理**：一次拖 13 個院區的檔案，自動依序處理
- **容錯規則**：自動辨識「減」、「檢」、「拿」、「電」、「初」等變體和錯字
- **跨裝置同步**：所有資料寫入 Google Sheet，任何裝置打開都看到同樣的儀表板
- **雙週分析儀表板**：院區 × 醫師 × 雙週的矩陣表，顯示週次與人次（去重病患）
- **本期 vs 上期**：自動比對前一個雙週區間，顯示變化量
- **拿藥獨立分類**：labeno 含「拿」會自動歸入「減重拿藥」虛擬醫師
- **匯出範本格式**：一鍵下載可貼進現有 Excel 報表的「各醫師含拿藥明細」

## 檔案說明

| 檔名 | 用途 |
|---|---|
| `index.html` | 主程式（櫃檯打開的網頁）|
| `AppsScript_backend.gs` | Google Apps Script 後端程式碼（要貼進 Sheet 的 Apps Script）|
| `setup_guide.html` | 首次設定的圖文教學 |
| `template_sheet.xlsx` | Google Sheet 資料庫格式範本 |

## 初次設定流程

1. 開一份新的 Google Sheet
2. 在 Sheet 內：擴充功能 → Apps Script
3. 把 `AppsScript_backend.gs` 整份貼進去
4. 執行 `setupSheets()` 建立工作表
5. 部署 → 新增部署作業 → 網頁應用程式 → 任何人可存取 → 部署
6. 把 Web App URL 寫進 `index.html` 裡的 `BUILTIN_ENDPOINT` 常數

詳細步驟請打開 `setup_guide.html` 參考。

## 篩選規則（超寬鬆，容忍懶散 key-in）

**保留條件**（三條同時成立）：
1. 含「減」或「檢」字（容錯：簡/鑑）
2. 結尾不是 `T00`
3. 不是純「減免」類型

**因此這些都會保留**：`減1W`、`減拿2`、`R減*2W`、`C M減1.3`、`S減2*`、`減R2W!`、`減c*2w`、`減C`、`C減`、`減拿`、`R減2周@`（即使沒有 W、用 * 代替、大小寫不一⋯都收）

**會排除**：`減免`、`減免T00`、`減重T00`、`PW*1`、`IM`、`NSUG2*10` 等

## 技術細節

- 純前端 HTML + JavaScript，無需安裝任何環境
- SheetJS 函式庫處理 Excel 檔（多 CDN 後援載入）
- 資料庫透過 Google Apps Script Web App
- localStorage 不再用來存資料，所有上傳紀錄改自 Sheet 同步

## 維護備註

未來修改 Apps Script 程式碼後，請用「**管理部署作業 → 編輯 → 新增版本 → 部署**」更新版本，這樣 URL 不會變、HTML 不用改。
